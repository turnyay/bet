# Bet

A Solana-based peer-to-peer betting platform.

## Project Structure

- `web/` - React web application

## Getting Started

### Web Application

1. Navigate to the web directory:
```bash
cd web
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The application will be available at [http://localhost:3042](http://localhost:3042)

## Features

- **Peer-to-Peer Betting**: Create and participate in bets directly with other users
- **Transparent & Secure**: All bets are recorded on the Solana blockchain
- **Fast & Low Cost**: Built on Solana for near-instant transactions and minimal fees
- **Automatic Settlement**: Smart contracts handle payouts automatically
